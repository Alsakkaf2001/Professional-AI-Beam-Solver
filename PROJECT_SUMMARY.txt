================================================================================
TECHNICAL SPECIFICATIONS
================================================================================

PHYSICS ENGINE:
- Solver Type: Euler-Bernoulli Beam Theory (2D frames)
- Stiffness Matrix: 6×6 per element with transformation
- Assembly: Global partition method with boundary conditions
- Solution: NumPy direct solver (np.linalg.solve)
- Numerical Precision: Double (float64)
- Singularity Detection: Condition number checking

ELEMENT CAPABILITIES:
- Element Type: 2D Straight Beam
- DOF: 3 per node (UX, UY, RZ)
- Stiffness: Axial + Flexural + Coupling
- Transform: Full 6×6 rotation + scaling

SUPPORT TYPES:
- Pin (2 DOF constrained)
- Fixed (3 DOF constrained)
- Roller-X (1 DOF: UY)
- Roller-Y (1 DOF: UX)
- Spring (Elastic - future)

LOAD TYPES:
- Point Load: Force (FX, FY) + Moment (MZ)
- Distributed Load: Uniform/Linearly varying
- Element loads converted to equivalent nodal forces

MATERIALS (Database):
- Steel: E=200 GPa
- Aluminum: E=70 GPa
- Concrete: E=30 GPa
- Custom materials supported

SECTIONS (Database):
- IPE: 100, 200, 300, ...
- HEB: 100, 200, ...
- Circular, Rectangular
- Custom sections supported

RESULTS:
- Node displacements (UX, UY, RZ)
- Support reactions (FX, FY, MZ)
- Element internal forces (N, V, M)
- Shear force diagrams (cubic interpolation)
- Bending moment diagrams (cubic interpolation)
- Deformed shape (Hermite interpolation)

================================================================================
EXAMPLE USAGE
================================================================================

PYTHON API:
```python
from solver import StandaloneBeamSolver

solver = StandaloneBeamSolver()
solver.create_model("My Beam")

# Add geometry
n1 = solver.add_node(0, 0)
n2 = solver.add_node(5000, 0)
solver.add_beam(n1.id, n2.id, material='steel', section='IPE-200')

# Boundary conditions
solver.add_fixed_support(n1.id)
solver.add_load(n2.id, fy=-50)

# Solve
results = solver.solve()
print(results['reactions'])
```

WEB API:
```bash
# Create model
curl -X POST http://localhost:5000/api/model/new \
  -H "Content-Type: application/json" \
  -d '{"name":"Structure"}'

# Add nodes and solve
curl -X POST http://localhost:5000/api/solve

# Export
curl -X POST http://localhost:5000/api/export/pdf \
  -H "Content-Type: application/json" \
  -d '{"filename":"report.pdf"}'
```
